Aspiring Adda | Career Pathways 2026
A modern, high-performance career exploration platform built with Vanilla JavaScript, Tailwind CSS, and Lucide Icons.

🚀 Features
Dynamic Roadmaps: Progress-tracked learning paths for AI, Web Dev, VLSI, and more.
Skill Assessments: Interactive quizzes for every roadmap step.
Career Quiz: A multi-stage assessment to match users with their ideal domain.
Job Board: Simulated job listings with skill-based filtering and "Save" functionality.
Dark Mode: Fully responsive design with glassmorphism UI.
🛠️ Tech Stack
HTML5 / CSS3 (Tailwind)
Vanilla JavaScript (State-based architecture)
Lucide Icons
